# Admin Dashboard UI

A modern, fully responsive Admin Dashboard UI built using **React + Vite** for the **Cerso Web Development Internship - Task 3**.

## 🚀 Live Demo

https://admin-dashboard-weld-zeta-81.vercel.app/

## 📌 Features

- Responsive Sidebar Navigation
- Top Header with Search Bar
- Notification Icon
- User Profile Section
- Dashboard Statistics Cards
- Monthly Revenue Line Chart
- Weekly Activity Bar Chart
- Recent Activity Table
- Quick Action Buttons
- Modern & Clean UI
- Responsive Design (Desktop, Tablet & Mobile)
- Dark Mode Support
- Smooth Hover Effects & Animations

## 🛠️ Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS3
- React Icons
- Chart.js / react-chartjs-2

```
src/

```

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/TG-Bharath-99/Admin-DashBoard.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
